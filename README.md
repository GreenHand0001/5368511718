# 项目简介

## 核心功能
- 基于YOLOv8的桥梁斜拉索缺陷检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 自定义置信度和IOU阈值
- 可选择显示标签和原图
- 包含1100张数据集，涵盖'Slippage'、'corrosion'、'crack'三类缺陷

## 适用场景
- 桥梁结构健康监测
- 斜拉索缺陷检测
- 基础设施安全评估
- 工程质量检测
- 研究与教学演示

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9、OpenCV、PyQt5、Ultralytics和PyTorch 1.9等技术栈
- 提供完整源码、数据集、训练好的模型及训练结果
- 包含UI界面源文件、图标和项目运行教程
- 支持多输入方式（单图片、文件夹、视频、摄像头）
